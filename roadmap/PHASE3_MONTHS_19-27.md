## CCSP Study Plan Details

### CCSP Domains Overview

1. **Domain 1** (17%): Cloud Concepts, Architecture, and Design
   - Cloud computing concepts and architectures
   - Design principles for cloud security
   - Cloud deployment and service models

2. **Domain 2** (20%): Cloud Data Security
   - Data classification
   - Data protection (encryption, DLP)
   - Data discovery and rights management

3. **Domain 3** (17%): Cloud Platform and Infrastructure Security
   - Infrastructure security (compute, storage, network)
   - Security controls and hardening
   - IaC security

4. **Domain 4** (17%): Cloud Application Security
   - Application security principles
   - API security
   - SDLC security integration

5. **Domain 5** (17%): Cloud Security Operations
   - Security operations (monitoring, incident response)
   - Disaster Recovery and Business Continuity
   - Forensics and investigation

6. **Domain 6** (12%): Legal, Risk, and Compliance
   - Legal and regulatory requirements
   - Risk management
   - Contracts and procurement
