# Phase 1: PAM + Kubernetes Mastery (Months 1-11)

**Duration**: 11 months (44 weeks)
**Weekly Commitment**: 10-12 hours/week
**Focus**: Master CyberArk PAM platform + Close Kubernetes hands-on gap

---

## Overview

### Primary Objectives

1. **Master CyberArk PAM Platform**
   - Install and configure all PAM components (Vault, CPM, PVWA, PSM)
   - Build enterprise PAM scenarios
   - Earn 3 CyberArk certifications

2. **Close Kubernetes Hands-On Gap**
   - 12 months continuous K8s practice (M1-11, extending through M12)
   - Deploy production-grade multi-tier applications
   - Master RBAC, networking, storage, security

3. **Build Foundation for Consulting**
   - Develop professional English skills (reading → writing)
   - Start monthly presentations (from Month 3)
   - Create first 3-4 portfolio projects

### Certifications Earned

- **CyberArk Defender** (Month 5) - Foundation level
- **CyberArk Sentry** (Month 8) - Advanced administration
- **CyberArk Guardian** (Month 11) - Expert level

See [docs/CYBERARK_CERTIFICATIONS.md](../docs/CYBERARK_CERTIFICATIONS.md) for detailed preparation guides.

### Success Metrics (by Month 11)

- ✅ All 3 CyberArk certifications obtained
- ✅ Can deploy production-grade K8s clusters
- ✅ 3-4 documented portfolio projects on GitHub
- ✅ Comfortable reading and writing technical English
- ✅ Delivered first consulting presentations
- ✅ Complete PAM lab environment operational

## Weekly Schedule Pattern

### Typical Week (10-12 hrs/week)

**Weekdays (Mon-Fri)**:
- Morning (30 min): English - Read technical docs + take notes
- Mid-day or Evening (1 hr): Technical learning (rotate: PAM focus 3 days/week, K8s 2 days/week)
- **Total weekday**: 1.5 hrs/day × 5 days = 7.5 hrs

**Weekends**:
- Saturday: Lab work (1.5 hrs)
- Sunday: Review + documentation (1 hr)
- **Total weekend**: 2.5 hrs

**Weekly Total**: 10 hrs/week (sustainable, family-friendly)

### Time Allocation by Activity

| Activity | Hours/Week | Percentage |
|----------|-----------|------------|
| Study/Reading | 3-4 hrs | 30% |
| Hands-On Labs | 6-7 hrs | 70% |
| English Learning | Integrated | Daily 30 min |
| Consulting Skills | 1 hr | Starting M2 |

## Key Success Factors

### 1. Sustainable Pace
- 10-12 hrs/week (not more, not less)
- Family-friendly schedule
- No burnout, consistent progress

### 2. Hands-On Focus
- 70% labs, 30% study
- Real deployments, not just theory
- Build muscle memory through practice

### 3. Continuous Kubernetes
- 12 months K8s practice (M1-11, extending to M12)
- Addresses hands-on gap completely
- Production-ready skills by Phase 2

### 4. Strategic Certification Timing
- Defender at M5 (after 4 months hands-on)
- Sentry at M8 (after 7 months hands-on)
- Guardian at M11 (after 10 months hands-on)
- Optimal spacing prevents cramming and burnout

### 5. Portfolio Building
- Start early (Month 4)
- Document everything
- Professional presentation on GitHub
